# 🏥 Hospital Management System

A full-stack web application built using Flask to manage hospital operations efficiently.

## 🚀 Features
- Patient Management
- Doctor Management
- Appointment Scheduling
- Billing System
- Reports & Dashboard

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite / MySQL

## ▶️ How to Run
1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py
